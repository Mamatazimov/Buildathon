# FaceID Monitor 👥

Yuzni aniqlash va real vaqt rejimida kelayotgan odamlarni identifikatsiya qilish tizimi. Loyiha ikki mustaqil qismdan iborat.

## Tizim arxitekturasi

### 1. AI & Face Recognition (Sun'iy intellekt qismi)
* **Vazifasi:** Kamera oqimidan yuzlarni aniqlash, ularni bazadagi yuzlar bilan solishtirish va odam kimligini aniqlash.
* **Texnologiyalar:** Python, OpenCV, Face-Recognition library.

### 2. Backend API
* **Vazifasi:** Tizim sozlamalari, foydalanuvchilar (yuz ma'lumotlari) bazasini boshqarish va aniqlangan odamlar tarixini saqlash.
* **Texnologiyalar:**  Python (FastAPI/Postgresql/pgvektor).
